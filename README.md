## JSON_Lua_Exchange

### Intro

It's a terminal tool of exchanging JSON and Lua, made with Python3.

### Usage

```bash
$ python3 [project directory/file], [JSON/Lua file path]
# Example
$ python3 . test.json # Convert to test.lua
$ python3 . test.lua # Convert to test.json
```
